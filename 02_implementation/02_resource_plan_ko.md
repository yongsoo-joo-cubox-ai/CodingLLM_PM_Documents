## 1. 프로젝트 개요

### 1.1 조직 구조 및 역할 분담

```
프로젝트 총괄 (PO): 문기봉 상무 (대보DX)
    │
    ├─ PoC/구현 파트: 대보DX
    │   └ 담당: 문기봉 상무 (단독)
    │   └ 역할: 프로토타입 개발, 기능 구현, 데모 시연
    │   └ 기간: ~3월 (PoC 완료까지)
    │
    └─ R&D/상용화 파트: 시선AI
        └ 서브PO: 주용수
        └ Phase 1 (1-3월): 서포트 역할 (인프라 + vLLM 배포 지원)
        └ Phase 2 (4월~): 본격 개발 투입
```
