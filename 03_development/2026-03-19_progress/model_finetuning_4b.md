## 1. 실험 목적

자연어 요구사항 → UASL YAML 명세 자동 변환을 위해 Qwen3-4B-Instruct 기반 LoRA 파인튜닝을 진행하였다. 이번 실험의 주요 목적은 **모델에 UASL 지식이 정상적으로 주입되는지 확인**하는 것이다.

---

## 2. 모델 구성

| 모델 | 입력 | 출력 |
|------|------|------|
| **Model1** | 자연어 요구사항 | Entity Specification (YAML) |
| **Model2** | 자연어 요구사항 + Entity Spec | IAS / SUIS / Workflow (YAML) |

---

## 3. 학습 데이터

| 항목 | Model1 | Model2 |
|------|--------|--------|
| Seed 데이터 | 20개 | 20개 |

- 모든 데이터는 3단계 검증기(L1/L2/L3) 통과 데이터만 사용
- 각 검증기는 UASL 스펙 명세 내용대로 구현

## 5. L3 미달 원인 분석

| 불일치 유형 | 비율 |
|------------|------|
| SUIS-IAS intent 불일치 | **56%** |
| SUIS 필드-Entity 속성 불일치 | 17% |
| Workflow-Entity 불일치 | 27% |
