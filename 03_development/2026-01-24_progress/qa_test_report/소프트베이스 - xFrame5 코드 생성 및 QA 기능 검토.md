#### 회의록 2026-01-22
1. 진행 경과 설명

- xFrame5 자료 변환 후 지식 DB화 완료

- 질의응답 기능 중 할루시네이션 문제 해결 완료

- 응답 품질 높이기 위한 개발 진행 중

  
2. 문의 사항

- 별도 서버 없이 Ollama로 사용자 PC에서 로컬 실행 가능한 질의응답 도구 가능한가?

- Ollama + Gemma3 1B로 자바/자바스크립트 코딩 답변 가능함 확인

=> 자바/자바스크립트 학습이 되어 있는 모델이기 때문에 경량이어도 가능한 것임

=> 당사도 별도 모델 개발이 로드맵에 있으나 단기 목표는 아님

=> xFrame을 별도 학습 시키는 것은 비용과 시간 상 무리이며 버전 변경 있을 때마다 학습시켜야 하는 문제

=> 당사가 추진하는 사업 방향은 xFrame에 국한되는 것이 아니며 범용적인 솔루션 추구 & 개발자 개인용이 아니라 프로젝트 및 전사 단위에서 공유하는 서버로 개발하는 중임

3. 요청 사항

- to 소프트베이스: 질의응답 샘플 20개 이상: 5개는 질의응답 전체 / 15개는 질의만

=> 내일까지 제공하기로 함

- to 시선AI: 향후 개발 일정 및 고객에게 보여줄 수 있는 xFrame5 관련 질의응답 작동 영상.
