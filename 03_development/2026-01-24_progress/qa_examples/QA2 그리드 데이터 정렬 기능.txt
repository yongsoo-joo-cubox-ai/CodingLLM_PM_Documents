다음은 xFrame5 그리드 데이터 정렬 기능에 대한 정리입니다.
(※ 멀티라인 그리드 / 트리 그리드 내용은 제외하고, 일반 Grid 기준으로 설명합니다.)

1. 관련 속성 (Property)

column_sort
팝업 메뉴에서 컬럼 정렬 기능 메뉴 표시 여부를 지정한다.

column_sort_disp
정렬 시 헤더에 표시되는 정렬 아이콘(삼각형)의 표시 방식을 설정한다.

column_multisort_disp
다중 정렬 시 정렬 순서 번호와 정렬 표시 아이콘의 표시 방식을 설정한다.

sorting (컬럼 데이터부 속성)
해당 컬럼의 정렬 가능 여부를 지정한다.

sort_datatype (컬럼 데이터부 속성)
정렬 시 사용할 데이터 타입(숫자/문자 등)을 지정한다.

soft_func / sort_func (컬럼 데이터부 속성)
사용자 정의 정렬 비교 함수를 지정한다.

2. 관련 API

sort(nColumn, nSortOrder)
단일 컬럼 정렬을 수행한다.

multisort()
현재 설정된 다중 정렬 정보를 기준으로 정렬을 수행한다.

multisortex(sortOpt, bToggle)
문자열 옵션을 이용해 다중 컬럼 정렬을 수행한다.

releasesort()
그리드의 모든 정렬 상태를 초기화한다.

getcolumnsortorder(nColumn)
지정한 컬럼의 정렬 상태를 반환한다.
(0: 미정렬, 1: 오름차순, 2: 내림차순)

getsortcolumns()
현재 정렬이 적용된 컬럼 인덱스 배열을 반환한다.

getmultisort()
현재 다중 정렬 상태 문자열을 반환한다.

setcolumnsorting(nColumn, bEnable)
특정 컬럼의 정렬 가능 여부를 설정한다.

setcolumnsortdisp(nMode)
컬럼 정렬 아이콘 표시 방식을 설정한다.

setcolumnsortfunc(nColumn, strFuncName)
특정 컬럼에 사용자 정의 정렬 함수를 지정한다.

getcolumnsortfunc(nColumn)
컬럼에 지정된 정렬 함수 이름을 반환한다.

3. 관련 이벤트

on_sortcomplete
그리드 정렬 처리가 완료된 후 발생한다.
정렬 시간 측정, 정렬 컬럼 정보 확인 등에 사용된다.

4. 샘플 소스
(1) 정렬 완료 이벤트 예제
// 그리드 on_sortcomplete 이벤트
function grd_on_sortcomplete(objInst, nSortStartTime, nSortEndTime)
{
	factory.consoleprint("정렬 처리 시간: " + (nSortEndTime - nSortStartTime) + "ms");
}

(2) 단일 / 다중 정렬 API 사용 예제
// "sort" 버튼 이벤트
function btn_sort_on_click(objInst)
{
	this.grd.sort(this.cbo_column.val(), this.cbo_sortorder.val());
}

// "multisort" 버튼 이벤트
function btn_multisort_on_click(objInst)
{
	this.grd.multisort();
}

// "multisortex" 버튼 이벤트
function btn_multisortex_on_click(objInst)
{
	var sort_opt, sort_opt_arr = [];

	sort_opt_arr.push("0:" + this.cbo_sortorder0.val());
	sort_opt_arr.push("1:" + this.cbo_sortorder1.val());
	sort_opt_arr.push("2:" + this.cbo_sortorder2.val());

	// sort_opt 형식: "0:ASC,1:DESC,2:TOGGLE"
	sort_opt = sort_opt_arr.join(",");

	this.grd.multisortex(sort_opt, false);
}

// "releasesort" 버튼 이벤트
function btn_releasesort_on_click(objInst)
{
	this.grd.releasesort();
}

(3) 사용자 정의 정렬 함수 예제
/**
 * 그리드 사용자 정의 정렬 함수
 */
function GridSortFunc(objInst, nColumn, nSortOrder, nDataType, strValueA, strValueB)
{
	var nValueA, nValueB;

	if (nDataType == XFD_GRID_DATA_NUMERIC) {
		nValueA = parseFloat(strValueA);
		nValueB = parseFloat(strValueB);

		if (nSortOrder == XFD_GRID_SORT_ASC) { return nValueA - nValueB; }
		else if (nSortOrder == XFD_GRID_SORT_DES) { return nValueB - nValueA; }
		else { return 0; }
	}
	else {
		if (strValueA == strValueB) { return 0; }
		if (nSortOrder == XFD_GRID_SORT_ASC) { return strValueA > strValueB ? 1 : -1; }
		else if (nSortOrder == XFD_GRID_SORT_DES) { return strValueA > strValueB ? -1 : 1; }
		else { return 0; }
	}
}


위 기능들을 활용하면

헤더 클릭 정렬,

단일/다중 컬럼 정렬,

정렬 상태 제어 및 초기화,

업무 규칙에 맞는 사용자 정의 정렬

을 xFrame5 Grid에서 모두 구현할 수 있다.
