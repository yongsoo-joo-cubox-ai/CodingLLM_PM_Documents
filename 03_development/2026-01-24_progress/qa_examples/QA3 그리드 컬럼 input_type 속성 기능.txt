다음은 xFrame5 그리드 컬럼 input_type 속성의 기능에 대한 설명입니다.
(일반 그리드 기준이며, 멀티라인 그리드 / 트리그리드는 제외합니다)

■ 관련 속성 / API / 이벤트
● 관련 속성

input_type
그리드 데이터부 컬럼에 표시되는 입력 방식(표현 방식)을 정의하는 속성

● 관련 API

getcolumninputtype(columnIndex)

setcolumninputtype(columnIndex, inputType)

getiteminputtype(rowIndex, columnIndex)

setiteminputtype(rowIndex, columnIndex, inputType)

● 관련 이벤트

on_itembtnclick
(input_type이 button 인 경우 발생)

■ input_type 기능 설명

input_type 속성은 그리드 컬럼의 셀을 어떤 UI 형태로 표시할지를 결정합니다.
컬럼 단위로 설정할 수 있으며, 필요 시 아이템(셀) 단위로도 재정의할 수 있습니다.

주요 input_type 값 예시
값	의미	설명
0	editbox	기본 입력 박스 형태
3	calendar	날짜 선택 캘린더
7	html	HTML 문자열 표시
9	button	버튼 형태 셀
10	image	이미지 전용 셀

컬럼에 설정된 input_type이 기본값이며
아이템에 input_type이 설정되면 아이템 값이 우선 적용됩니다.

아이템 input_type이 -1이면 컬럼의 input_type을 따릅니다.

editable 속성과 함께 사용 시 입력 가능/불가능 여부가 제어됩니다.

■ 관련 샘플 소스
컬럼 input_type 변경 예제
// "setcolumninputtype" 버튼 이벤트
function btn_setcolumninputtype_on_click(objInst)
{
	var input_type;

	input_type = this.grd.getcolumninputtype(0);

	// input_type이 editbox인 경우, calendar로 변경
	if (input_type == 0) {
		this.grd.setcolumninputtype(0, 3);
		this.grd.setheadertext(1, 0, "calendar");
	}
	// input_type이 editbox가 아닌 경우, editbox로 변경
	else {
		this.grd.setcolumninputtype(0, 0);
		this.grd.setheadertext(1, 0, "editbox");
	}

	// 변경된 컬럼 input_type 표시
	this.grd.refresh();
}


아이템(셀) 단위 input_type 변경 예제
// "setiteminputtype" 버튼 이벤트
function btn_setiiteminputtype_on_click(objInst)
{
	var input_type;

	// 아이템별 input_type을 구함
	input_type = this.grd.getiteminputtype(1, 0);

	// 아이템별 input_type 미지정 상태인 경우, 컬럼의 input_type을 구함
	if (input_type == -1) { input_type = this.grd.getcolumninputtype(0); }

	// input_type이 editbox인 경우, calendar로 변경
	if (input_type == 0) {
		this.grd.setiteminputtype(1, 0, 3);
	}
	// input_type이 editbox가 아닌 경우, editbox로 변경
	else {
		this.grd.setiteminputtype(1, 0, 0);
	}
}


button 타입 컬럼 예제 (input_type = 9)
/**
 * 그리드 컬럼 유형이 button인 경우, 버튼 클릭 이벤트 처리
 */
function grd_on_itembtnclick(objInst, nClickRow, nClickColumn)
{
	factory.consoleprint("grd_on_itembtnclick event");
	screen.alert("Button Click, Row = " + nClickRow + ", Column = " + nClickColumn);
}


image 타입 컬럼 동작 설명

input_type = 0 (editbox)
→ 텍스트 + 이미지 함께 표시

input_type = 10 (image)
→ 셀 데이터는 이미지 정보로 처리, 이미지만 표시

input_type = 10 인 경우에만
컬럼 데이터부 Image 그룹 속성이 적용됨

필요하시면 input_type 값 전체 목록,
또는 특정 타입(button / image / html) 상세 동작도 추가로 설명해 드릴 수 있습니다.
